# Examples — a FAKE candidate and a FAKE job

Everything in this folder is **invented**. "Jordan Rivera", "Northwind Logistics", and "Lumen
Robotics" are fictional — no real person or company. The point is to show the *shape* of the inputs and
outputs without anyone supplying real data.

| File | What it is |
|---|---|
| `sample-candidate.md` | a fake résumé-as-text (the kind of thing you'd paste into `prep-intake`) |
| `sample-jd.md` | a fake job description for the role being targeted |
| `sample-profile-and-gaps.md` | a sample *generated* artifact — the profile + fit/gap map `prep-intake` would write to a real user's `workspace/` |

When you run the kit for real, artifacts like the third file land in your **gitignored** `workspace/`,
built from *your* real materials — never committed to a shared repo.
