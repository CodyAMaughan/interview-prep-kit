# Code of Conduct

## Our pledge

This project exists to help people prepare for interviews honestly and to support job-seekers in a
tough market. We pledge to make participation a harassment-free experience for everyone, regardless of
background or experience level.

## Our standards

Examples of behavior that contributes to a positive environment:
- Being kind, patient, and respectful — many people here are job-hunting and under stress.
- Giving and gracefully accepting constructive feedback.
- Focusing on what is best for the community and for the people this tool serves.

Unacceptable behavior includes:
- Harassment, insults, or derogatory comments.
- Publishing others' private information (this project is built around *not* leaking personal data —
  hold that standard with each other too).
- Contributing anyone's real personal data, or repurposing the tool as a live-interview "cheat" aid.

## Enforcement

Maintainers may remove, edit, or reject contributions that violate this Code of Conduct. To report an
issue, open a GitHub issue or contact the maintainer.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.
